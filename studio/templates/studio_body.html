We provide Android application solutions for all kind of project, from web or iOS existing project adaptation to devices specific feature based project new project.

One App to rule them all

We provide solutions for all the existing Android form factor devices, from wearables to tablets. Let's the Android fragmentation hurdle in a Far Far Away realm as the Android framework empowers us to develop one app to rule them all.

Android Pure

All our Android solutions are exclusively developed with the Android official framework.

It means:

  • No overlay API which could possibly impact performance and maintainability while limiting device capabilities,
  • No restriction concerning embedding all Android and Google APIs, i.e. connectivity (wifi, bluetooth, nfc), sensors (touch, GPS, gyroscope...), camera features, location services.

The whole Android playground is our playground!

Unleash the power

We feature native app development to access another performance level. Let's speak with C++ and OpenGL ES words to fulfill the more demanding high power requirements.

3D Real-Time Rendering (RTR) is at heart of iGLou Studio proficiency. We provide custom made graphic solutions for mobile and desktop platform, from simple real-time render integrated component to stand-alone app powered by state-of-the-art render engine.

Integrated or stand-alone

RTR based solutions can be considered,

  • Either as an integrated component, part of a bigger app, like a 3D object viewer in a catalog,
  • Either as a stand-alone application featuring a more immersive graphics content, like games.

We could provide both.

Native OpenGL ES or licenced Unreal Engine

Simple rendering could be implemented natively by an in-house OpenGL ES render. Main benefit is no restriction on software distribution aspect.

For advanced rendering requirements, we provide Unreal Engine based solution. It is basically on of the most advanced available game engine featuring state-of-the-art rendering techniques.

Targeted platform

  • ANDROID
  • iOS
  • LINUX
  • OS X
  • WINDOWS
  • WEB

As a mobile app without internet service is like a car without fuel or GPS device without map, we provide our own custom cloud service support, going with app development. It's a flexible, scalable and secure mean to interact with your app user.

Cloud Package

Our cloud service offer include all the mandatory steps to provide a full cloud-based app experience providing

  • An App/server interface, managing request and data feed-back,
  • A data management service through cloud-based database.
  • A Web administration interface, allowing custom data visualisation.

Our solution is based on the very robust and powerful Django web framework.

Server Management

Beside software services we offer server management services featuring

  • A server management plan, including hosting, update and maintenance
  • A server automated deployment service.

You could get a full-stack cloud service completely adapt for your need.
